# 🎮 PokéBattle

Application full-stack pour tracker les combats Pokémon Champions.

## Déploiement Rapide

1. **Frontend** → Vercel : https://vercel.com
2. **Backend** → Render : https://render.com  
3. **Database** → MongoDB Atlas : https://www.mongodb.com/cloud/atlas

## Features

- ✅ 23 Pokémon Champions
- ✅ Suivi des victoires/défaites
- ✅ Données persistantes
- ✅ Interface mobile-friendly
- ✅ Gratuit ($0/mois)
